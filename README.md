# HST-More-Language

为Hades-StarTool提供翻译文件

## 语言

[中文](https://) | [EN](https://) |

## 如何使用翻译文件

将所需的文件添加进 /storage/emulated/0/HadesTool/Language 目录下

或将其完全替换为本项目的Language文件夹

添加后可在应用内自动识别或手动切换

目录结构:

- /storage/emulated/0/HadesTool
  - resAccount
  - Language
    - zh.json
    - en.json
    - (其它翻译文件)
  - rubbish


## 添加本地化

以json格式修改文本值,将本地化的翻译文件上传至本项目.

也可以将翻译文件发送给我

在应用下次更新时可能将你的本地化翻译文件内置
